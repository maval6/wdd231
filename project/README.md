# Global Gateway Travel Services

A comprehensive travel information website designed and developed by Massango V. Moukouelle for the WDD231 course.

## Project Overview

Global Gateway Travel Services is a responsive website providing international travelers with essential information, resources, and tools. The site features destination spotlights, travel planning tools, visa information lookup, and currency conversion functionality.

## Features

- Responsive design for mobile and desktop devices
- Dynamic destination content loaded from JSON data
- Interactive travel planning tools with localStorage integration
- Currency converter
- Visa requirement lookup
- Weather information for popular destinations

## Technologies Used

- HTML5 for semantic structure
- CSS3 for styling and responsive design
- JavaScript ES6+ with modules
- JSON for data storage
- LocalStorage for client-side data persistence

## Pages

1. **Home Page (index.html)** - Main landing page with featured destinations and service overview
2. **Destinations (destinations.html)** - Explore global destinations with filtering capabilities
3. **Travel Resources (resources.html)** - Tools and resources for trip planning

## Setup and Usage

1. Clone the repository
2. Open the project in a web browser
3. Navigate through the pages using the navigation menu
